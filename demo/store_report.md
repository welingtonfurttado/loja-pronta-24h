## Como Usar Esta Demo

### Para Vender Lojas Prontas para Clientes

1. **Apresentação:** Use o `index.html` como demonstração visual da loja. Abre no browser sem precisar de servidor.

2. **Personalização:** Para adaptar a um cliente específico:
   - Substitua as cores no `:root` do CSS (`--dark`, `--red`, `--yellow`)
   - Atualize os produtos no `catalog.json` com os produtos reais do cliente
   - Regenere as páginas de produto com `_run_demo_store_builder.py`

3. **Proposta Comercial:** Use os dados do `store_report.md` para embasar a proposta:
   - Mostre as projeções de receita com diferentes verbas
   - Destaque o score da loja (100/100)
   - Apresente o checklist de itens implementados como entregável

4. **Preço Sugerido:** Uma loja com essa estrutura pode ser vendida entre R$ 1.500 e R$ 5.000 dependendo do cliente.

5. **Recorrência:** Ofereça manutenção mensal (R$ 300–800/mês) incluindo:
   - Atualização de produtos e preços
   - Gestão de campanhas de tráfego pago
   - Relatório mensal de performance
